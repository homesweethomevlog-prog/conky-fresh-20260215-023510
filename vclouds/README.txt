Put your VClouds weather icon PNG files in this folder:

~/.config/conky/vclouds

Conky will auto-pick icons by weather condition using these preferred names:

clear: sun.png, sunny.png, clear.png, 32.png
partly: partlycloudy.png, partly_cloudy.png, 30.png
cloudy: cloudy.png, cloud.png, 26.png
rain: rain.png, showers.png, rainy.png, 11.png, 12.png
storm: storm.png, thunderstorm.png, thunder.png, 04.png
snow: snow.png, sleet.png, 16.png
fog: fog.png, mist.png, 20.png

Notes:
- PNG format only.
- Keep transparent backgrounds for best look.
- The weather source is BBC and icon condition key is generated by weather_bbc.py.
